Unit 5: Introduction to Web Security

Web Security is a core concept in modern cybersecurity. In this unit, we discuss the basic definition, why it matters, and how it is used in real systems.

Subtopics:
- Web Security definition and core ideas
- Common use cases of Web Security
- Challenges and limitations

Short explanation:
Web security focuses on protecting websites and web applications from attacks like XSS and SQL injection.

Example questions:
1. Define Web Security and explain its purpose.
2. Give two real-world scenarios where Web Security is used.
3. What are some challenges when implementing Web Security in practice?
